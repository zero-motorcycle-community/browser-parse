Zero Log File Parser

Decoder for Zero motorcycle MBB or BMS log files. Offline HTML decoder converts binary log files to viewable and downloadable text. Use Chrome or Firefox.



Online parser Use this if you don't need offline parsing or are having trouble downloading.

Instructions

1. Download the latest version from GitHub with the button above
2. Move the zero-log-parser.html file or just open it from your downloads folder
3. Use the Zero Motorcycles app to get your bikes log files, see screen shot below
4. Log files can be sent as an email attachment or saved to Google Drive
5. Save the sent Zero log files to your downloads folder or other location
6. Open the zero-log-parser.html file in Chrome or Firefox
7. Choose the .bin log file to parse, view and download as text
